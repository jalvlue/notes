- Server
	- 连接器: 管理连接, 验证权限
	- 缓存器: 查询语句如果命中则直接返回结果
	- 分析器: 词法分析, 语法分析
	- 优化器: 生成执行计划, 选择索引
	- 执行器: 操作引擎, 返回结果
- 存储引擎: 实际对数据进行存储, 向上对执行器提供读写接口, 插件式的, 支持多种不同的引擎
	- `innodb`: 默认
	- `myisam`
	- `memory`
